# Página de la tienda de vinilos 'Disco'

![Logo](./imgs/logo.jpg)

## About
**Disco** es un proyecto encargado para la asignatura de diseño web, es una landing page para una tienda de vinilos y mercancía retro. Su lema es **_Tu lo buscas, nosotros lo encontramos._**

## Finalidad del proyecto

* ### Responsive Design
Se busca una página que pueda ser accesible desde cualquier pantalla.
* ### Practicar
Practicar conceptos aprendidos sobre el lenguaje de estilos CSS y crear documentos HTML con una correcta estructura semantica.
* ### Aprender
Con este proyecto se busca tambien el aprendizaje, al encontrar retos en la elaboración del sitio en la busqueda de soluciones se logra aprender nuevas formas y metodos de resolver conflictos.
